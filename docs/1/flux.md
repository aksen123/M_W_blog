## Fulx 패턴이란!?

플럭스 패턴은 애플리케이션의 상태와 데이터 흐름을 관리하는 방법을 정의하는 구조적 접근 방식이다. 주요 특징은 데이터의 흐름이 단방향으로 이루어진다는 것인데 
이로 인해 데이터의 변경이 명확하게 추적 가능하고, 애플리케이션의 상태가 예측 가능하게 된다.
